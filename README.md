# Msc thesis battery surrogate
Repository for the master's thesis: "Neural Network Enhanced Physics-based Surrogate Model Framework for Multiphysics Battery Simulations"

**Note:** The code base was developed from experimental work and remains in the experimental stages.

## Navigating the repository
The repo consists of four main directories regarding 
1. data acquisition and processing ([data](data) & [data_pybamm](data_pybamm)), 
2. neural network model training and evaluation ([nn_model](nn_model)), 
3. and lastly symbolic regression optimization and evaluation ([sr_model](sr_model)). 

Each of which contains their own `README.md` files describing intended use.
